# README(2021.1.2)

## 小程序操作逻辑

1. 首先获取用户授权：get userinfo(app.js的global data)
2. 再获取用户基本信息:以ifexist全局变量标志

获取这两个全局变量是进去后续操作的逻辑基础。如果不获取用户授权和基本信息按道理就不能进行其它操作。

## 抱团回家功能Pages说明

### index

用户进入小程序的第一个首页，可以进行修改增加功能实现

### fillinfo

完善用户信息，目前用户信息的字段只有

1. 姓名
2. 学号
3. 联系方式
4. 校区
5. 宿舍

可以按功能需要删改

> 除上述意外，其余额外功能与抱团回家其它功能都独立。
